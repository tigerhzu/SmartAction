# SmartAction Help Center

SmartAction Help Center 是設定、維護與故障排除的入口。

## Help Center 項目

- Quick Start / 快速教學：App 內建的短版使用教學。
- Full Documentation / 完整文件：開啟線上完整文件。
- Open GitHub Repository：開啟 SmartAction GitHub 專案。
- Report Issue：回報問題或提出改善建議。
- About SmartAction：顯示版本、作者與 App 描述。

## 本機文件

- `docs/quick-start.md`
- `docs/client-workspace.md`
- `docs/firefox-container-helper.md`
- `docs/help.md`
