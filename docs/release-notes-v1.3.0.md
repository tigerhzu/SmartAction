# SmartAction v1.3.0

SmartAction v1.3.0 focuses on visual quality, day-to-day reliability, and
sharing the same setup across different Windows computers.

## Highlights

- Ten visually distinct Ring themes with dedicated materials, particles,
  hover animation, click feedback, and reduced-motion support.
- Animated Ocean waves and sharks, spectral Halloween mist, Sakura water and
  petals, HUD-style Cyber segments, crystal Purple, Frozen ice, flowing Lava,
  Cosmic orbits, mature Tiger energy, and elastic Kawaii syrup.
- Twelve selectable zodiac constellations with configurable line colour.
- Drag the Ring clockwise or counter-clockwise without launching an action.
- Settings, PowerShell Library, and Client Workspace open on the monitor where
  the Ring was invoked.
- Client Workspace supports folders and drag-to-reorder clients.
- Classic, Cute, and Woven Light global interfaces with immediate preview.
- Cute includes a bundled high-resolution default background, image cropping,
  positioning, zoom, and custom-image support.
- The Windows tray and application logo now follow the active global
  interface style.
- Native Windows hotkey recovery for long-running background sessions.
- Reduced Emoji catalog memory use by keeping only default skin-tone entries.
- Import and export complete SmartAction profiles.

## Performance and accessibility

- Procedural Ring effects are kept within the 60 FPS frame budget on the
  development test machine.
- Inactive visual assets are loaded lazily.
- Reduced Motion disables or freezes non-essential Ring animation.
- Theme labels and dialogs use stronger contrast on both light and dark
  backgrounds.

## Windows package

Download `SmartAction-Release-v1.3.0.zip`, extract the whole folder, then run
`install.bat` or `SmartAction.exe`.

Do not download the GitHub source-code ZIP as the Windows application package.
